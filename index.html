Fast Food Picker

This very simple project is comprised of an UNO, LCD, and button. On each button click the device gives the user a suggestion on where they should eat. This project was inspired by the countless times my girlfriend and I have been at an impass on where to eat due to our indecicesvness. Problem Solved!

Digital Height Reader

This little machine is meant to be places at a fixed height, 78 inches in my case, allowing it's user to stand under it for a quick and easy measurmnent of their height. This prototype device uses an arduino uno, a generic ultrasonic sensor, a buzzer all powered by a 9 volt battery or power suply. The purpose of the buzzer is so the user knows when the reading has been made and it is safe to walk away from the sensor to see the reading. Sometimes, if the user is not directly under the sensor, there will be a bad reading which notifies the user to try again.
